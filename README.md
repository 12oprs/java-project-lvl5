### Task Manager  

[Link to application (heroku.com)](https://project5-4321.herokuapp.com/)  
Documentation is located on `/swagger.html` when app is running  
  
Simple Task Manager on Spring Framework with Postgres database.  
Application allow to create tasks with statuses and labels. Set executors for task.  
Features:
- login/password authorization
- filtering tasks by status, executor, author, label
- api documentation generated by Swagger
- error tracking (plugged to rollbar.io)
